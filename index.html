A dataset of images from the Simpsons was trained on DCGAN (a machine learning algorithm).
The algorithm squashes the images down to represent them with 128 numbers.
It then generates new images that were numerically similar to the given data.

The boxes above represent the values of each of the 128 numbers that correspond to the image on the left
Select one or many of the boxes, and use the UP and DOWN keys on your keyboard to increase/decrease their values
You might need to wait a minute for results.
Change the 'delta' value to control the effect of each keyboard press. (between 0 and 1)

made with ml5.js and p5.js
